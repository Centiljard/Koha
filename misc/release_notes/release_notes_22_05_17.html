RELEASE NOTES FOR KOHA 22.05.17

28 Nov 2023

Koha 22.05.17 is a bugfix/maintenance release.

It includes 4 enhancements, 6 bugfixes, and 2 security bugfixes.

System requirements

You can learn about the system components (like OS and database) needed for running Koha on the community wiki.

Security bugs

  • 35290 SQL Injection vulnerability in ysearch.pl
  • 35291 File Upload vulnerability in upload-cover-image.pl

Bugfixes

Architecture, internals, and plumbing

Critical bugs fixed

  • 34959 Translator tool generates too many changes

Other bugs fixed

  • 32978 'npm install' fails in ktd on aarch64, giving unsupported architecture error for node-sass
  • 35024 Do not wrap PO files

MARC Authority data support

Critical bugs fixed

  • 33404 Authorities imported from Z39.50 in encodings other than UTF-8 are corrupted

OPAC

Other bugs fixed

  • 33848 Enabling Coce in the OPAC breaks cover images on bibliographic detail page

Tools

Critical bugs fixed

  • 26611 Required match checks don't work for authority records

    Sponsored by Waikato Institute of Technology

    This fixes match checking for authorities when importing records, so that the required match checks are correctly applied. Previously, match checks for authority records did not work.

Enhancements
