REST API

Enhancements

  • 33036 REST API: Merge biblio records

    A new endpoint of REST API /biblios, to merge two bibliographic records. You need to pass parameters with a json file. Complete endpoint: <base_url>/api/v1/biblios/<biblo_id>/merge Parametes of json file:

    • biblio_id_to_merge (mandatory)
    • rules (optional)
    • framework_to_use (optional)
    • datarecord (optional) More info in the Swagger/OpenAPI Specification of the API

    Sponsored by Technische Hochschule Wildau
